# YouTube Channel Analysis
Beginner-level data analysis project using Python and Pandas to explore the most subscribed YouTube channels.

## Tools
- Python
- Pandas
- Plotly
- Jupyter / Google Colab

## What’s inside
- Data cleaning
- Subscriber distribution analysis
- Category-wise and year-wise visualizations

⚠️ Plotly charts may not render properly on GitHub.  
Run the notebook locally or in Google Colab for full output.
